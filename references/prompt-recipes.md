# 提示词模板

用这份文档把模糊需求整理成能直接喂给供应商接口的提示词。

## 基础结构

如果用户没有直接给出成熟提示词，优先使用这个结构：

```text
目标：<这张图要拿来做什么>
主体：<主要对象>
风格：<照片、插画、海报、3D 等>
构图：<镜头角度、取景、留白、版式>
光线：<光源、氛围、时间>
色彩：<配色方向>
细节：<材质、纹理、配件、环境>
文字（逐字）："<如果有精确文案就写这里>"
约束：<必须保留、必须避免、不要水印、不要多余肢体等>
```

如果用户本身已经写得很具体，就做规范化整理，不要无意义扩写。

## 高价值模板

### 海报或社媒创意

可用结构：

```text
为 <品牌/产品> 制作强视觉广告海报。
主视觉主体：<主体>。
视觉风格：<风格>。
版式：为标题和 CTA 预留清晰空间。
光线与氛围：<光线>。
色彩方向：<配色>。
文字（逐字）："<精确文案>"。
约束：高对比、层级清晰、不要水印、除引号内文案外不要出现乱码文字。
```

### 产品图

可用结构：

```text
<产品> 的高端产品摄影。
场景设计：<台面、道具、环境>。
镜头：<特写/俯拍/三分之四视角>。
光线：<棚拍或自然光方案>。
重点细节：<玻璃冷凝、水珠、磨砂纸、拉丝金属等>。
约束：包装比例真实、不要多余物体、不要文字变形。
```

### 概念图

可用结构：

```text
<主体/场景> 的概念设计图。
世界观细节：<时代、材质、建筑、服装、天气>。
构图：<大全景、角色设定图、环境关键帧>。
氛围：<情绪>。
渲染风格：<厚涂、动画、电影感、风格化 3D>。
约束：结构合理、轮廓清晰、不要水印。
```

### 改图

可用结构：

```text
编辑提供的图片。
只修改：<明确改动内容>。
保持不变：<人物身份、姿势、服装、版式、产品形状、背景元素等>。
匹配原图：<光线/透视/颗粒/风格>。
约束：不要新增物体、不要改动文字、不要改变整体构图。
```

每次继续迭代改图时，都要重复写一次“不变量”。

## 精确文字处理

- 把引号内文案单独放一行，例如：`文字（逐字）："..."`。
- 明确说明文字位置和风格。
- 如果文案很长、结构复杂或包含多语言，提前说明“先出背景图、后期再排字”通常更稳。
- 中文海报场景尽量使用短句、强层级，并明确换行方式。

## 参考图使用方式

如果用户提供多张图片，在构造提示词前先给它们分配角色：

- `参考图 1：主体身份`
- `参考图 2：色彩方案`
- `参考图 3：构图参考`
- `参考图 4：材质或渲染风格`

不要把多张参考图混成一句模糊描述，要明确每张图各自提供什么信息。

## 迭代规则

- 每次只改一个维度：构图、光线、颜色、表情或文字布局。
- 已经成功的约束在每次重试时都保留。
- 如果第一版结果结构就错了，重写整段提示词。
- 如果第一版已经接近目标，只收窄差异，不要完全推翻重来。
